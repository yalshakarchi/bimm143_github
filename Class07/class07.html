Hierarchical Clustering

The main function for Hierarchical Clustering is called hclust(). Unlike kmeans (which does all the work for you) you can’t just pass hclust() our raw input data. It needs a “distance matrix” like the one returned from the dis() function.

d <- dist(x)
hc <- hclust(d)
plot(hc)

To extract our cluster membership vector from a hclust()result object we have to “cut” our tree at a given height to yeild separate “groups”/ “branches”.

plot(hc)
abline(h= 8, col="red", lty=2)

To do thus we use the cutree() function on our hcluster() object:

grps <- cutree(hc, h=8)
